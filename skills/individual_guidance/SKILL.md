# 技能：个性化科研指导

## 技能名称
`individual_guidance` — 学生个性化科研指导

## 核心目标
根据每位学生的能力画像、成长趋势和近期表现，为导师生成针对性的培养建议和发展方案。帮助导师在有限的精力下，为每位学生提供高质量、有针对性的指导。

## 触发条件
当用户发出以下类型的请求时激活本技能：
- "帮我看看某某同学该怎么指导"
- "给某某同学提点培养建议"
- "某某同学下一步应该重点发展什么"
- "为某某同学制定一个本学期的培养计划"
- 包含关键词：个性化指导、发展建议、培养方案、指导意见、能力提升

## 输入格式
```json
{
  "student_id": "string (必填)",
  "focus_area": "string (可选，指定重点关注的方面)",
  "time_horizon": "string (可选，'short'=本月, 'medium'=本学期, 'long'=全年)",
  "context": "string (可选，导师补充的背景信息，如近期观察)"
}
```

## 输出格式
```json
{
  "student_summary": "对学生当前状态的简要描述",
  "strengths": [
    {"dimension": "代码实现", "evidence": "近期完成了XX项目的核心模块开发", "score": 90}
  ],
  "growth_areas": [
    {"dimension": "学术写作", "current_level": "基础", "target_level": "熟练", "gap_analysis": "..."}
  ],
  "action_plan": [
    {
      "priority": 1,
      "action": "每周精读一篇本领域顶会论文并撰写阅读笔记",
      "dimension": "论文阅读",
      "expected_duration": "4周",
      "milestone": "完成4篇精读笔记，能在组会中清晰分享"
    }
  ],
  "encouragement": "总结性鼓励话语",
  "next_check_in": "建议的下次评估时间"
}
```

## 执行步骤

### 第一步：获取学生画像
调用 `student_progress` 技能获取学生的完整能力数据，包括：
- 当前各维度评分（雷达图数据）
- 近6个月的成长趋势
- 近期里程碑事件
- 与上一评估周期的对比

### 第二步：分析优势与短板
基于能力数据进行分析：
- **优势识别**：得分超过75的维度，尤其是持续上升的维度
- **短板识别**：得分低于55的维度，或停滞/下降的维度
- **潜力评估**：虽然当前得分不高但近期上升趋势明显的维度

### 第三步：考虑上下文因素
综合考虑以下因素调整建议：
- 学生年级（研一侧重基础，研二侧重创新，研三侧重产出）
- 研究方向对各能力的不同要求
- 近期事件（如即将开题、投稿deadline等）
- 导师补充的观察和背景信息

### 第四步：制定行动计划
为每个需要提升的维度制定具体、可执行的行动方案：
- 每条建议都要具体到行为级别（如"每周读一篇论文"而非"多读论文"）
- 设定清晰的时间节点和可衡量的里程碑
- 优先级排序：先解决最关键的1-2个方面
- 确保行动计划的工作量合理，不要过多

### 第五步：生成鼓励性总结
用温暖、鼓励的语气写一段总结：
- 肯定学生的努力和进步
- 指出最值得期待的成长方向
- 表达对学生未来发展的信心

## 注意事项

### 教育理念
- 每位学生都有自己的节奏，不要强行对标"标准模板"
- 发展建议应当基于学生的实际情况，而非理想化目标
- 鼓励"优势发展"策略，不必追求全面满分
- 同时不忽视明显的短板，尤其是影响毕业和产出的维度

### 语言风格
- 使用中文，语气温和、鼓励、具体
- 用"建议"而非"必须"，用"可以尝试"而非"应该"
- 引用具体事件和数据作为支撑（如"你上个月完成的XX项目说明..."）
- 避免空泛的鼓励（如"继续加油"），替换为具体的肯定

### 频率与时机
- 建议每月至少生成一次个性化指导
- 重要节点（开题、中期、投稿前）主动生成
- 如果学生连续两个月某维度显著下降，触发预警

### 与其他技能的配合
- 可结合 `collaboration_recommend` 推荐合适的学习伙伴
- 可结合 `literature_review` 推荐适合该生阅读的论文
- 行动计划中的里程碑可自动关联到 `student_progress` 的事件追踪

## 关联技能
- `student_progress`：提供能力数据（必须先调用）
- `collaboration_recommend`：推荐学习伙伴
- `literature_review`：推荐适合阅读的文献
- `meeting_record`：从会议记录中获取学生表现信息
